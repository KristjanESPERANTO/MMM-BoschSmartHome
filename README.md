# MMM-BoschSmartHome (Beta)

Click here for the Magic Mirror [Forum Thread](https://forum.magicmirror.builders/topic/14347/mmm-bsh-bosch-smart-home/)

`git clone https://github.com/jalibu/MMM-BoschSmartHome.git`
